关于Shellshock bash bug

#安全  #Shellshock  #bash 
“不小心”看到了shellshock bash bug 不明觉厉，按图索骥： $env x=‘() { :;}; echo vulnerable’ bash -c ‘echo this is a test’ If you see the word “vulnerable” as an answer, your system is, well… vulnerable. 中枪。。。 立刻update+upgrade，再回过头看看到底啥情况： Your Bash shell is simply running more code after a function (the “() { :;};” part), and that shouldn’t be happening. The function is the “allowed” code, while everything after it is where the potentially “malicious” code could be installed.
