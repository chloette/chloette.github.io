Book list

2014-12-23 — Written by Guo Yi
#记录 

####2014⁄2015

######在看 >《设计心理学2》

《设计模式》

######想看 >《算法导论》

######已看 （按时间逆序）

  • 《CMOS图像传感器集成电路 –原理、设计及应用》 >科普：如何从“光线”生成“raw buffer图像信息”

  • 《少妇白洁》（18禁） >以女性视角展开的成人小说

  • 《三体》 >地球-太阳系-银河系-宇宙；欲罢不能的看完全集；面对宇宙，无知的人类，是多么渺小。

  • 《影响力》（在这里看的“压缩版”~） >潜移默化的心理学

  • 《囚徒健身》 >这节奏完全停不下来~

  • 《ios核心动画高级技巧》 > IOS动画“入门”可以快速浏览的一本书；前五章了解结构（其实UI框架都差不多），后几章熟悉技巧；翻译的很多地方值得商榷（但翻译很辛苦，值得认可）

  • 《设计心理学》（The Design of Everyday Things） >深层次回答，设计的What,Why,How问题

  • 《别让我思考》（Don’t make me think） >用序中作者老婆的话来概括（大意）即：I won’t visit it if it’s complicated。

  • 《商战》 > 最后的最后，还是一个度的抉择问题。标准回答是：It depends，不是么。

  • 《随遇而安》 >一个人生流水账

  • 《打造Facebook 亲历Facebook爆发的5年》 > 副标题很好的诠释了本书内容

  • 《我的抗战》 > 换个角度看那段历史

  • 《浪潮之巅 第二版》 >各种大公司的八卦，因果分析略有马后炮的感觉。BTW，个人觉得本书最关键的不是告诉你这些公司为何成功/失败的，而是告诉你，这些巨头，一样是由普通人运作的。而普通人，就一定会有精明或犯浑的时候，巨头也一样。

  • 《Linux+树莓派 玩转智能家居》（Smart Home Automation with Linux and Raspberry Pi） > Geek向，帮助扩宽智能家居技术思路的一本书

  • 《1Q84》 >很独特的写作风格，一如小小人的故事，让人欲罢不能。

  • 《别让猴子跳回背上》（Monkey Business） >很容易上手的管理书籍，一下午就可以翻完，然后就去调教猴子吧。

  • 《黑客与画家》 >付出=回报 -> 加倍的付出=加倍的回报；所以，如果想获得爆发式的增长（回报），就要做好（使用）爆发式的投入。

  • 《失控 全人类的最终命运和结局》 (Lost control) >略浮夸的副标题，但蜂群（扁平化）/自制这些贯穿全文的主旨会表明，这也许没错。读起来费力，却值得。
